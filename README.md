# Feetal E-commerce Sales Dashboard

## Overview
This dashboard provides an interactive and visual representation of Feetal's e-commerce sales data. It helps stakeholders analyze performance across different dimensions such as time, geography, product categories, and payment modes.

## Features
- **Dynamic Filters:** Filter data by Quarter and State.
- **Key Metrics:**
  - Total Sales Amount
  - Total Profit
  - Total Quantity Sold
  - Average Order Value (AOV)
- **Visual Insights:**
  - Top Buyers
  - Sales by State
  - Monthly Profit Trends
  - Quantity Distribution by Category
  - Quantity Distribution by Payment Mode
  - Profit by Sub-category

## KPIs
- **Sum of Amount:** 438K
- **Sum of Profit:** 37K
- **Sum of Quantity:** 5615
- **Sum of AOV:** 121K

## Charts & Visualizations
- **Bar Charts:** For Top Buyers, State-wise Amount, Monthly Profit, Sub-category Profit.
- **Donut Charts:** For Category-wise Quantity and Payment Mode distribution.
- **KPIs:** Displayed at the top for quick reference.

## Usage
1. Select the desired **Quarter** and **State** using filters.
2. Analyze KPIs for overall performance.
3. Drill down into charts for detailed insights.

## Purpose
This dashboard is designed for:
- **Business Analysts** to identify trends and anomalies.
- **Sales Teams** to track performance by region and category.
- **Management** for strategic decision-making.

## Technology
- Built using **Power BI**
- Data Source: E-commerce transactional data.
